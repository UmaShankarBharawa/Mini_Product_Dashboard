# Mini Product Dashboard

A **Mini Product Dashboard** built with **React.js**
It fetches product data from a public API and displays it in a responsive grid with search, filter, and sorting functionality.

## Tech Stack

- React.js
- Font Awesome for icons

## Features

- Fetch products from [Fake Store API](https://fakestoreapi.com/products)
- Display products in a responsive grid
- Search products by name/title
- Filter products by category
- Sort products by price (Low to High / High to Low)
- Responsive design for desktop, tablet, and mobile
- Error handling component if API fails
- Light/Dark theme toggle

## Getting Started

```bash
git clone https://github.com/umashankarbharawa/mini_product_dashboard.git
cd mini_product_dashboard
npm install
npm run dev
```
